# Abora Calamares Config

This folder contains the Abora branding and settings package for the Calamares GUI installer.

It is intended to ship in the live ISO alongside the local `calamares` package so the
`Install Abora OS` launcher can open a real graphical installer instead of a terminal wrapper.

The remaining work is validation: confirm the packaged Calamares build starts cleanly,
uses the Abora branding, and completes an end-to-end install.
